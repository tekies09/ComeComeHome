## 개요

**프로젝트 목적 및 용도는 다음과 같다.**
- 암기를 요구하는 컴퓨터 프로그램 게임 개발
- 간편하게 누구나 즐길수 있도록 프로그램 구현 
- 기억의 미로 게임을 오마주하여 구현
